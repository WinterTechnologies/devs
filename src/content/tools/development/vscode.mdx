Visual Studio Code is a free, open-source code editor developed by Microsoft.
It's the most popular code editor for developers, and is available for Windows,
macOS, and Linux. It's also available as a web app, which means you can use it
from your browser without installing anything.

Using tools like [rojo](https://rojo.space) you can easily create and manage
roblox games using source control, and using [roblox-ts](https://roblox-ts.com)
you can even use JSX to write Roact UI.
